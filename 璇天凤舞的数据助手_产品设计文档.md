# 璇天凤舞的数据助手 - 产品设计文档

## 一、项目概述

### 1.1 产品定位
【璇天凤舞的数据助手】是一款面向二级市场投资者的量化策略展示工具，帮助用户查看、回测和分析量化投资策略的表现，辅助投资决策。

### 1.2 目标用户
- 初级投资者：了解量化投资概念，学习基本策略
- 中级投资者：对比不同策略表现，优化投资组合
- 专业投资者：深度分析回测数据，挖掘投资机会

### 1.3 核心价值
- 透明展示量化策略的历史表现
- 多维度分析策略的收益与风险
- 提供实用的投资工具辅助决策

## 四、数据指标体系

### 4.1 策略评价指标

| 指标名称 | 计算公式 | 说明 | 评价标准 |
|----------|----------|------|----------|
| 年化收益率 | (总收益/年) × 100% | 策略平均年收益 | 越高越好 |
| 夏普比率 | (策略收益-无风险收益)/策略波动率 | 风险调整收益 | > 1 为好 |
| 最大回撤 | (最高点-最低点)/最高点 | 历史最大亏损 | 越小越好 |
| 胜率 | 盈利次数/总交易次数 | 交易成功率 | 越高越好 |
| 波动率 | 收益标准差 | 收益稳定性 | 适中为佳 |
| 卡玛比率 | 年化收益/最大回撤 | 收益回撤比 | 越高越好 |

### 4.2 风险等级划分

| 等级 | 描述 | 最大回撤 | 夏普比率 |
|------|------|----------|----------|
| R1 | 极低风险 | < 5% | > 2.0 |
| R2 | 低风险 | 5-10% | 1.5-2.0 |
| R3 | 中等风险 | 10-20% | 1.0-1.5 |
| R4 | 较高风险 | 20-30% | 0.5-1.0 |
| R5 | 高风险 | > 30% | < 0.5 |

## 六、技术建议

### 6.1 技术栈
- **框架**：Taro 3.x（多端统一开发）
- **状态管理**：Redux / MobX
- **图表**：ECharts（适配小程序）
- **数据**：预留 API 接口，后期对接后端

### 6.2 数据来源（后期）
- 实时行情：需要对接行情数据 API
- 策略数据：后端回测引擎生成
- 用户数据：本地存储 + 云端同步

### 6.3 开发优先级
1. **Phase 1**：首页 + 策略详情页（核心展示）
2. **Phase 2**：策略广场 + 筛选功能
3. **Phase 3**：我的策略 + 数据工具
4. **Phase 4**：个人中心 + 高级功能
